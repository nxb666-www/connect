## 四、后端服务启动

### 1. 编译项目

```bash
cd 社交平台目录/backend
mvn clean install -DskipTests
```

### 2. 启动顺序

**必须按以下顺序启动**（网关依赖其他服务注册到 Nacos）：

```bash
# Java 路径（根据实际安装位置修改）
JAVA="C:/Program Files/Java/jdk-17/bin/java.exe"

# 1. 启动用户服务 :8081
cd social-user
$JAVA -jar target/social-user-1.0.0.jar

# 2. 启动动态服务 :8082
cd ../social-post
$JAVA -jar target/social-post-1.0.0.jar

# 3. 启动社交关系服务 :8083
cd ../social-social
$JAVA -jar target/social-social-1.0.0.jar

# 4. 启动消息服务 :8084
cd ../social-message
$JAVA -jar target/social-message-1.0.0.jar

# 5. 启动搜索服务 :8085
cd ../social-search
$JAVA -jar target/social-search-1.0.0.jar

# 6. 启动统计服务 :8086
cd ../social-statistics
$JAVA -jar target/social-statistics-1.0.0.jar

# 7. 启动 AI 服务 :8087
cd ../social-ai
$JAVA -jar target/social-ai-1.0.0.jar

# 8. 最后启动网关 :8080
cd ../social-gateway
$JAVA -jar target/social-gateway-1.0.0.jar
```

### 3. 验证服务启动

访问 Nacos 控制台 http://localhost:8848/nacos ，在"服务管理 → 服务列表"中应看到 8 个服务全部注册成功。

## 七、常见问题

### 1. 服务启动失败：端口被占用

```bash
# 查找占用端口的进程
netstat -ano | findstr "端口号"

# 结束进程
taskkill /PID 进程ID /F
```

### 2. Nacos 连接失败

确保 Nacos 已启动且端口 8848 可访问。检查各服务的 `bootstrap.yml` 中 Nacos 地址是否正确。

### 3. 数据库连接失败

确认 MySQL 服务已启动，检查 `application.yml` 中的数据库连接信息（地址、用户名、密码）。

### 4. 前端请求 401

JWT 令牌过期（24小时有效期），重新登录即可。

### 5. AI 对话超时

AI 服务已配置 60 秒超时。如仍超时，检查网络是否能访问 https://open.bigmodel.cn 。

### 6. 图片无法显示

确认图片上传目录存在且有写入权限。消息服务负责静态文件托管。

## 八、快速启动脚本（Windows）

创建 `start-all.bat` 一键启动所有后端服务：

```bat
@echo off
set JAVA=C:\Program Files\Java\jdk-17\bin\java.exe
set BASE=社交平台目录\backend

echo Starting social-user...
start "social-user" "%JAVA%" -jar "%BASE%\social-user\target\social-user-1.0.0.jar"
timeout /t 10

echo Starting social-post...
start "social-post" "%JAVA%" -jar "%BASE%\social-post\target\social-post-1.0.0.jar"
timeout /t 10

echo Starting social-social...
start "social-social" "%JAVA%" -jar "%BASE%\social-social\target\social-social-1.0.0.jar"
timeout /t 10

echo Starting social-message...
start "social-message" "%JAVA%" -jar "%BASE%\social-message\target\social-message-1.0.0.jar"
timeout /t 10

echo Starting social-search...
start "social-search" "%JAVA%" -jar "%BASE%\social-search\target\social-search-1.0.0.jar"
timeout /t 10

echo Starting social-statistics...
start "social-statistics" "%JAVA%" -jar "%BASE%\social-statistics\target\social-statistics-1.0.0.jar"
timeout /t 10

echo Starting social-ai...
start "social-ai" "%JAVA%" -jar "%BASE%\social-ai\target\social-ai-1.0.0.jar"
timeout /t 10

echo Starting social-gateway...
start "social-gateway" "%JAVA%" -jar "%BASE%\social-gateway\target\social-gateway-1.0.0.jar"

echo All services started!
pause
```
